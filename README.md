## 功能介绍（完整见运行截图）
管理员：基本功能包括登录、注册和修改密码。可以进行学生管理、报名记录管理、准考记录管理、身份认证管理和系统配置。能够审核学生报名信息，管理考试信息并支持信息导入导出。此外，还可通过客服聊天功能进行用户支持和查询反馈。
